## ADDED Requirements

### Requirement: Listen mobile responsive buttons
Các nút đáp án trong game listen PHẢI fill đầy chiều rộng màn hình trên thiết bị có width <= 480px.

#### Scenario: Listen buttons on mobile
- **WHEN** người dùng mở listen game trên thiết bị có width <= 480px
- **THEN** các nút đáp án hiển thị full width, xếp theo 1 cột dọc

#### Scenario: Listen buttons on desktop
- **WHEN** người dùng mở listen game trên thiết bị có width > 480px
- **THEN** các nút đáp án hiển thị theo grid 2 cột như hiện tại
